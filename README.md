# scaffolder-vue3-vite-capacitor

Just git clone and enjoy Capacitor 3.0.0, Vue 3 and Vite. Make awesome Apps 

```bash
git clone https://github.com/jjlmoya/scaffolder-vue3-vite-capacitor.git
```


## dev:
```bash
npm run dev
```

## add platforms:
```bash
npm run platforms
```

## Sync App
```bash
npm run apps
```
